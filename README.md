# pytermcanvas

## Requirements
* terminal emulator (with [true color support](https://gist.github.com/sindresorhus/bed863fb8bedf023b833c88c322e44f9)) 
* python3
  * pip

## Install
```sh
pip install pytermcanvas
```
## Usage
```python
import pytermcanvas as ptc

canvas = ptc.TerminalCanvas(25, 10)
```
